Mushroomy's Island

Description

Mushroomy's Island is a 2-player collabrative game that requires one person to select the assets and other person to use the body posture to build a personalized garden. A character called 'Mushroomy' owes this garden and she will guide the guests to put the plants in the proper areas.

Game Mechanism

Microsoft Kinect will detect the human body posture and mimic similar shape of the plants into the screen. If the assets are set in a wrong place, mushroomy will say 'no'.
